# profileProject_IBM

Single-page portfolio website built as part of the **IBM Full Stack Software Developer** program. The site showcases professional profile, technical skills, IBM Cloud projects, and interactive recommendations.

## Live Demo

**[View LiveDemo](https://a422789.github.io/profileProject_IBM/)**

## Features

- **Responsive navigation** — Smooth scroll links to About Me, Skills, Projects, and Recommendations
- **About Me** — Personal introduction and profile image
- **Skills section** — HTML, CSS, JavaScript, Java, React, and Node.js with experience levels
- **Projects** — Highlights IBM Cloud–related work including microservices, Watson AI, and Cloudant
- **Recommendations** — Display existing testimonials and add new ones dynamically via JavaScript
- **Interactive form** — Submit recommendations with a success popup confirmation

## Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, Safari)

### Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/A422789/profileProject_IBM.git
   cd profileProject_IBM
   ```
2. Open `index.html` in your browser — no build step or server required.

### Optional: Local Server

For a closer-to-production experience, serve the folder with a simple HTTP server:

```bash
# Python 3
python -m http.server 8000
```

Then visit `http://localhost:8000`.
